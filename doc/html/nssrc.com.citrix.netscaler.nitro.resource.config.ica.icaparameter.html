icaparameter module¶

class icaparameter¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for Config Parameters for NS ICA resource.

enablesronhafailover¶

Enable/Disable Session Reliability on HA failover. The default value is No.<br/>Default value: NO<br/>Possible values = YES, NO.

hdxinsightnonnsap¶

Enable/Disable HDXInsight for Non NSAP ICA Sessions. The default value is Yes.<br/>Default value: YES<br/>Possible values = YES, NO.

l7latencyfrequency¶

Specify the time interval/period for which L7 Client Latency value is to be calculated. By default, L7 Client Latency is calculated for every packet. The default value is 0.<br/>Default value: 0<br/>Maximum length = 60.

builtin¶

Indicates that the ICA parameter is a built-in (SYSTEM INTERNAL) type.<br/>Possible values = MODIFIABLE, DELETABLE, IMMUTABLE, PARTITION_ALL.
