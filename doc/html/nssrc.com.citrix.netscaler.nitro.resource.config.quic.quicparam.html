quicsecrettimeout¶

Rotation frequency, in seconds, for the secret used to generate address validation tokens that will be issued in QUIC Retry packets and QUIC NEW_TOKEN frames sent by the Citrix ADC. A value of 0 can be configured if secret rotation is not desired.<br/>Default value: 3600<br/>Maximum length = 31536000.
