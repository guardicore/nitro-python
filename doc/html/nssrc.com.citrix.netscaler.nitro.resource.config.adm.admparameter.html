admserviceconnect¶

Parameter to enable/disable Citrix ADM Service Connect. This feature helps you discover your Citrix ADC instances effortlessly on Citrix ADM service and get insights and curated machine learning based recommendations for applications and Citrix ADC infrastructure. This feature lets the Citrix ADC instance automatically send system, usage and telemetry data to Citrix ADM service. View here [https://docs.citrix.com/en-us/citrix-adc/13/data-governance.html] to learn more about this feature. Use of this feature is subject to the Citrix End User ServiceAgreement. View here [https://www.citrix.com/buy/licensing/agreements.html].<br/>Default value: ENABLED<br/>Possible values = ENABLED, DISABLED.

lowtouchonboard¶

ENABLED value indicates that this ADC can be on-boarded onto Citrix ADM Service using auto-onboarding process. Refer to [https://docs.citrix.com/en-us/citrix-application-delivery-management-service/service-connect-onboarding.html] for further details.<br/>Possible values = ENABLED, DISABLED.
